Outside Projects

Application of Genetic Algorithms for Image Generation

This project was started originally for my Artificial Intelligence Practicum. Our goal was to compare different forms of local search. Specifically, we looked at hill stepping and genetic algorithms in the setting of procedurally generating images using polygons. For class, we implemented the project in Python. To see our paper, please click here. Unfortunately, Python was too slow for our needs. Thus, after the class was over, we implemented the algorithms in C++. This is still a work in progress. The repo can be found here.
